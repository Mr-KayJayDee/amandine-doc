# commandchannel

> Allows you to configure the bot's command channel.

<br>
## Usage :
*commandchannel #channel
## Examples :
*commandchannel add #bot-commands,
<br>*commandchannel remove #staff-commands,
<br>*commandchannel reset
## Aliases :
channelcmd,
<br>cmdchannel
