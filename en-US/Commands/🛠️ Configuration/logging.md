# logging

> Allows you to define which log you want to enable or disable.

<br>

## Usage :

*logging <setting> <on/off>

## Examples :

*logging messageUpdate on,
<br>*logging guildUpdate off,
<br>*logging all on,
<br>*logging channel #channel,
<br>*logging off
