# setjoin

> Allows you to configure the server welcome message and channel.

<br>
## Usage :
*setjoin <option> <argument> 
## Examples :
*setjoin on,
<br>*setjoin off,
<br>*setjoin reset,
<br>*setjoin channel <mention>,
<br>*setjoin message <message>,
<br>*setjoin options
## Aliases :
configjoin,
<br>joinconfig
