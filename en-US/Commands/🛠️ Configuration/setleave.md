# setleave

> Allows you to configure the server's goodbye message and channel.

<br>

## Usage :

*setleave <option> <argument> 

## Examples :

*setleave on,
<br>*setleave off,
<br>*setleave reset,
<br>*setleave channel <mention>,
<br>*setleave message <message>,
<br>*setleave options
