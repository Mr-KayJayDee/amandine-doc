# inviteblacklist

> Permet de blacklister un membre du système d'invitations.

<br>

## Usage :

*inviteblacklist <option> [@membre]

## Exemples :

*inviteblacklist add @member,
<br>*inviteblacklist remove @member,
<br>*inviteblacklist list

## Alias :

invbl,
<br>blinv
