# setjoin

> Permet de configurer le message et canal de bienvenue du serveur.

<br>
## Usage :
*setjoin <option> <argument> 
## Exemples :
*setjoin on,
<br>*setjoin off,
<br>*setjoin reset,
<br>*setjoin channel <mention>,
<br>*setjoin message <message>,
<br>*setjoin options
## Alias :
configjoin,
<br>joinconfig
