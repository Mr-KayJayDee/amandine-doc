# logging

> Permet de définir quels logs vous voulez activer ou non.

<br>

## Usage :

*logging <paramètre> <on/off>

## Exemples :

*logging messageUpdate on,
<br>*logging guildUpdate off,
<br>*logging all on,
<br>*logging channel #channel,
<br>*logging off

## Alias :

logs,
<br>logger
