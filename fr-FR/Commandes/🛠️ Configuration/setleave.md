# setleave

> Permet de configurer le message et canal d'au revoir du serveur.

<br>

## Usage :

*setleave <option> <argument> 

## Exemples :

*setleave on,
<br>*setleave off,
<br>*setleave reset,
<br>*setleave channel <mention>,
<br>*setleave message <message>,
<br>*setleave options

## Alias :

configleave,
<br>leaveconfig
