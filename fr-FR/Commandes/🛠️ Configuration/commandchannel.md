# commandchannel

> Permet de configurer le canal de commandes du bot.

<br>

## Usage :

*commandchannel #canal

## Exemples :

*commandchannel add #bot-commands,
<br>*commandchannel remove #staff-commands,
<br>*commandchannel reset

<br>cmdchannel
