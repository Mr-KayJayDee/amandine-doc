# setprefix

> Permet de configurer le préfixe du bot sur le serveur.

<br>

## Usage :

*prefix <nouveau préfixe>

## Exemples :

*prefix !!,
<br>*prefix ?

## Alias :

prefix

Robot: Gérer les messages
